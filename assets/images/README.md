# Imágenes recomendadas

Coloca aquí los archivos de imagen del proyecto. Usa formatos modernos (WebP preferido, JPG como fallback).

## Imágenes necesarias antes de publicar

| Archivo | Dimensiones | Uso |
|---------|-------------|-----|
| `og-image.jpg` | 1200×630 px | Open Graph / Twitter Card (vista previa en redes) |
| `hero-author.jpg` | 480×480 px | Foto del autor para la sección "Sobre mí" (opcional) |
| `favicon.png` | 32×32 y 192×192 px | Favicon del navegador |
| `apple-touch-icon.png` | 180×180 px | Icono en iOS al guardar en pantalla de inicio |

## Recomendaciones

- Optimiza todas las imágenes antes de subir: usa [Squoosh](https://squoosh.app) o similar
- Usa el atributo `alt` descriptivo en todo `<img>`
- Para imágenes de cursos/productos, crea una versión WebP + JPG y usa `<picture>` como elemento
- Evita imágenes de stock genéricas de finanzas (billetes, gráficas de trading) — no transmiten confianza

# Iconografía recomendada: Lucide Icons

Este proyecto usa **[Lucide Icons](https://lucide.dev)** como sistema de iconos.

## Cómo usar Lucide

### Opción A — CDN (más sencillo para empezar)
```html
<script src="https://unpkg.com/lucide@latest/dist/umd/lucide.min.js"></script>
<!-- Usar en HTML: -->
<i data-lucide="trending-up"></i>
<!-- Activar en JS: -->
<script>lucide.createIcons();</script>
```

### Opción B — SVG inline (mejor rendimiento, sin dependencia externa)
Descarga solo los SVG que necesites desde lucide.dev y pégalos directamente en el HTML.

## Iconos recomendados para este proyecto

| Icono | Uso sugerido |
|-------|-------------|
| `trending-up` | Crecimiento, rendimiento positivo |
| `shield-check` | Seguridad, confianza |
| `book-open` | Guías, cursos |
| `calculator` | Herramientas de cálculo |
| `star` | Ratings, valoraciones |
| `check-circle` | Pros de un curso |
| `x-circle` | Contras de un curso |
| `external-link` | Enlaces a Hotmart (afiliado) |
| `mail` | Newsletter, contacto |
| `info` | Notas informativas |
| `alert-triangle` | Avisos de riesgo |
| `chevron-down` | Acordeón FAQ |
| `menu` | Hamburger en mobile |

## Tamaños estándar del proyecto
- Iconos en línea con texto: 16px (1rem)
- Iconos en cards o destacados: 24px (1.4rem)
- Iconos decorativos grandes: 32px (2rem)
